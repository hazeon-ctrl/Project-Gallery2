Ambient Spectre

Audio Spectrum Shooter - Version 72 Design Specification
Date: 2025-10-21
Status: Stable, Feature-Complete
Framework: Vanilla HTML5 + Web Audio
API

1. Executive Summary
Version 72 represents a stable, playable arcade shooter where combo-driven power-ups and homing projectiles create fast-paced, rewarding gameplay. The game uses the D Dorian scale as its harmonic foundation, with all sound effects tuned to create a cohesive musical experience. The core loop centers on collecting pellets to activate stacking power-ups that dramatically increase player capabilities.
2. Core Mechanics
2.1 Player

Movement: WASD or Arrow keys (5 units/frame)
Shooting: Hold SPACE to fire

Base fire rate: Every 15 frames
Combo fire rate: Every 5 frames (when any power-up active)

Homing shots: ALL player projectiles have homing behavior (0.08 strength)
Starting health: 100 HP
Visual: 30x30 green square with cyan cockpit detail, engine glows, subtle green radial gradient aura

2.2 Enemies

Spawn system: Dynamic spawning based on enemies destroyed

Max enemies: 1 + floor(enemiesDestroyed / 3), capped at 5
1% chance per frame to spawn when below max

Behavior: Random vertical movement with shooting AI
Health: 100 HP each
Shoot interval: 120 frames
Reward: 50 points on destruction
Visuals: 30x30 squares with red/orange gradients, angular design, weapon ports, red radial aura

2.3 Pellets

Spawn rate: Every 45 frames
Max on screen: 16
Spawn area: Player's half of screen, avoiding danger zone
Value: 10 points each
Power-up trigger: Every 4 pellets collected
Visual: Pulsing colored circles with glow rings

2.4 Danger Zone

Location: Bottom 120 pixels of screen (audio spectrum visualization)
Damage: 3 HP every 30 frames when player inside
Healing mode: When "Spectrum Heal" power-up active, heals 1-4 HP/tick instead
Visual: Red spectrum bars (default) or green spectrum bars (heal mode)

3. Power-Up System (Stacking Combos)
3.1 Power-Up Types

Spread Shot - Fire 3-6 bullets in a spread pattern (increases with level)
Shield - Block chance: 30%/60%/90% by level (radius 50px, flashes white when blocking)
Spectrum Heal - Converts danger zone to healing zone, 1-4 HP/tick by level
Damage Boost - Adds +5/+10/+15 damage per level (visual: red indicator bars)

3.2 Stacking Mechanics

Multiple active: Can have all 4 power-ups simultaneously
Level stacking: Getting same power-up increases level (‚òÖ‚òÖ‚òÖ max)
Duration: 400 frames per power-up
Auto rapid-fire: Having ANY power-up active triggers rapid fire mode
Visual indicators: Stars (‚òÖ) show level, unique visual effects per type

3.3 Combo Benefits

Base fire rate: 15 frames
Combo fire rate: 5 frames (3x faster)
Homing missiles become devastating with spread shot
Multiple level-3 power-ups = screen-clearing power

### 4.2 Ambient Foundation
- **Drone:** Two sine waves (D3 + A3) at 0.06 gain
- **Purpose:** Establishes tonal center (root + fifth)
- **Routing:** Through reverb ‚Üí analyser ‚Üí destination

### 4.3 Sound Events

#### Player Shoot
- **Note:** A4 (normal) or G4 (spread shot)
- **Waveform:** Triangle (softer than square)
- **Duration:** 120ms with smooth envelope
- **Filter:** Low-pass at note*3 for warmth
- **Harmonic:** Subtle octave-up sine wave
- **Routing:** Through reverb

#### Enemy Shoot
- **Notes:** E3, G3, B3, or C4 (per enemy personality)
- **Waveforms:** Sawtooth, Square, Triangle, or Sine
- **Duration:** 100ms
- **Sub-harmonic:** Octave below for m
#### Hit Sound
- **Notes:** C4 ‚Üí F3 descent (two detuned sawtooths)
- **Sub-bass:** D3*0.5 for impact
- **Filter:** 400Hz ‚Üí 80Hz sweep
- **Duration:** 200ms
- **Routing:** Through delay + analyser

#### Collect Sound
- **Arpeggio:** D5 ‚Üí E5 ‚Üí G5
- **Heal chime:** A5 (subtle, delayed)
- **Waveform:** Sine
- **Duration:** 150ms per note
- **Routing:** Through reverb

#### Power-up Sounds
- **Spread Shot:** E4-G4-B4 (minor triad)
- **Shield:** F4-A4-C5 (subdominant)
- **Spectrum Heal:** G4-B4-D5 (major sound)
- **Damage Boost:** D4-A4-D5 (power chord)
- **Waveform:** Triangle
- **Duration:** 400ms
- **Routing:** Through reverb

#### Combo Level-Up
- **Note:** D5 * 2^(level*0.25) - ascending per level
- **Waveform:** Sine
- **Duration:** 200ms

#### Explosion
- **White noise:** Filtered 2000Hz ‚Üí 50Hz over 500ms
- **Chord cluster:** B3-C4-D4 (dark, dissonant)
- **Duration:** 300-500ms
- **Routing:** Through delay + reverb (big tail)

#### Enemy Spawn
- **Arpeggio:** Root note √ó [1.0, 1.2, 1.5, 1.8] (minor third, fifth, minor seventh)
- **Waveform:** Matches enemy timbre
- **Duration:** 240ms total

## 5. Visual System

### 5.1 Canvas Specifications
- **Size:** 800x600 pixels
- **Background:** Dark blue (#0d0d1f) with 25% opacity trail effect

- **Smoothing:** 0.8

- **Boundary line:** Dashed, color-coded

### 5.3 Visual Indicators

- **Shield:** Cyan ring (50px), flashes white when blocking, thickness = level
- **Damage Boost:** Red bars on ship side (count = level)

## 6. Game Balance

### 6.1 Difficulty Scaling
- Enemies spawn faster as you destroy more
- Max 5 simultaneous enemies
- Enemy density creates natural difficulty curve

### 6.2 Reward Loop
```
Collect 4 pellets ‚Üí Power-up ‚Üí
Rapid fire + Homing ‚Üí Destroy enemies ‚Üí
More pellets spawn ‚Üí Collect more ‚Üí
Stack power-ups ‚Üí Screen-clearing power

6.3 Risk/Reward

Pellets spawn in player area but near danger zone
Venturing into danger zone risky but necessary for pellets
Spectrum Heal inverts risk (danger zone becomes healing sanctuary)

7. Technical Implementation

7.1 Architecture

Single HTML file (~1000 lines)
No external dependencies (pure Web Audio API)
Event-driven game loop using requestAnimationFrame
Audio scheduled using AudioContext.currentTime

7.2 Performance Characteristics

Target: 60 FPS
Typical load: 10-15 active audio nodes
Memory: Minimal (no large buffers, all procedural synthesis)

7.3 Browser Compatibility

Requires Web Audio API support
Click-to-start required (browser autoplay policies)
Include cool 'start-up' sound to initiate game play

Tested on modern Chrome/Firefox/Safari

9. Design Philosophy

Core Tenets:

Audio-first design - Sound informs gameplay, not just decorates
Emergent complexity - Simple systems combine for depth
Rewarding skill - Homing + combos = power fantasy
Musical coherence - All sounds harmonize (D Dorian framework)
Visual clarity - Spectrum display both functional and aesthetic

Player Experience Goals:

Feel powerful when combos active
Hear progression through harmonic richness
Clear audio feedback for all actions
Risk/reward tension (danger zone)
Satisfying destruction (explosions with reverb tail)
